CSS (Cascading Style Sheets) é a linguagem que controla a apresentação visual dos elementos HTML. Ele permite definir cores, fontes, tamanhos, posicionamento, animações e efeitos para tornar a interface mais atraente e funcional.

Além de estilizar páginas de forma consistente, o CSS é essencial para criar layouts adaptáveis (responsivos), garantindo que o site funcione bem em diferentes dispositivos e tamanhos de tela.
