HTML (HyperText Markup Language) é a base de qualquer página web, sendo responsável por estruturar todo o conteúdo visível e invisível de um site. Ele define a hierarquia de elementos como títulos, parágrafos, listas, links e imagens, usando tags semânticas que facilitam a leitura tanto para usuários quanto para mecanismos de busca e tecnologias assistivas.

Um HTML bem estruturado contribui para a acessibilidade e a manutenção do código.
