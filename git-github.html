Git é um sistema de controle de versão que permite acompanhar todas as mudanças feitas em um projeto, facilitando o trabalho em equipe e a correção de erros. Ele cria um histórico completo do código, possibilitando voltar a versões anteriores e criar diferentes ramificações (branches) para testar novas ideias sem comprometer o projeto principal.

O GitHub, por sua vez, é uma plataforma online que hospeda repositórios Git e fornece ferramentas para colaboração, revisão de código e integração com outras ferramentas de desenvolvimento.
